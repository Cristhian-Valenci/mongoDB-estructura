# mongoDB-estructura

El  objetivo de este repositorio era, aprender a modelar base de datos no SQL, para ello, tenemos dos ejercicios en el nivel 1, en donde simularemos la base de datos de una optica, en el ejercicio 1 con la interfaz grafica desde el punto de vista de un cliente, y en el ejercicio 2 desde el punto de vista de las gafas.
Se entrega un archivo .json para cada collection de cada base de datos.
